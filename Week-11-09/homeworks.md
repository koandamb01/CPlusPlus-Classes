C++ Basic Functions Homework Exercises

1. Write a function ```add(int a, int b)``` that returns the sum of two integers
2. Write a function ```square(int n)``` that returns the square of a number
3. Write a function ```circleArea(double radius)``` that returns the area of a circle ```(π × r²)```
4. Write a function ```average(double a, double b)``` that returns the average of two numbers
5. Write a function ```power(int base, int exponent)``` that calculates base^exponent (assume positive exponent)
6. Write a function ```isPositive(int n)``` that returns true if the number is positive
7. Write a function ```isVowel(char c)``` that returns true if the character is a vowel (a, e, i, o, u)
8. Write a function ```countVowels(string str)``` that counts vowels in a string
9. Write a function ```isAdult(int age)``` that returns true if age is 18 or older
